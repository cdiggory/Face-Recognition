The ** Smart Attendance System** is a desktop application developed in Python that automates attendance management using Face Recognition technology. The system allows administrators to register students, capture facial data, train a recognition model, and automatically mark attendance through a webcam.
The application uses OpenCV for image processing, the Face Recognition library for feature extraction, and the K-Nearest Neighbors (KNN) algorithm for student identification. Attendance records are stored in CSV format and can be viewed or exported for reporting purposes.
**Features**
Student Registration with Face Capture
Automatic Student ID Generation
Face Recognition-Based Attendance
KNN Machine Learning Model
Attendance Record Management
CSV and Excel Export
Student Database Management
Real-Time Webcam Detection
User-Friendly Tkinter GUI
Configurable System Settings
**Technologies Used**

OpenCV
Face Recognition

Project Objective
To automate attendance tracking, reduce manual effort, prevent proxy attendance, and improve accuracy using facial recognition and machine learning techniques.
